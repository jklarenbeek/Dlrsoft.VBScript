This project contains some ASP Classic Compiler Samples Pages

Folder File Comments
Components IIS 5 Platform SDK Sample. Need to install the COM components. Components like BrowserCap will not work because it requires the COM+ comtext to accesses the Request object.
FileSystem.asp Works without registering any component. Need to grant ASP worker process account read/write access to the directory.
ServerVariables.asp Iterate through Request.ServerVariables
SessionRead.asp Shows all current session variables with sessionid.
SessionWrite.asp Show and manipulate all current session variables with sessionid.
SessionTest.asp Test whatever.
Database IIS 5 Platform SDK Sample. Some the Access samples tested. Make sure that the OLEDB provider is installed. Need read/write access to the access.mdb in the directory.
SimpleQuery.asp Using the local database Authors.mdb via ADODB. Showing a simple select query.
AddDelete.asp Using the local database Authors.mdb via ADODB. Adding and then deleting a record.
Update.asp Using the local database Authors.mdb via ADODB. Updates an author.
LimitRows.asp Using the local database Authors.mdb via ADODB. Shows Paging with authors table.
MultiScrolling.asp Using the local database Authors.mdb via ADODB. Shows Paging with scrolling on authors table.
Blob.asp Using the local sqlserver instance with the pubs database via ADODB. Showing logo of pub_info ONLY if database service is available!
StoredProcedures.asp Using the local sqlserver instance with the pubs database via ADODB. Call storedprocedure byroyalty ONLY if database service is available!
Simple IIS 5 Platform SDK Sample. All the samples tested.

test Some small samples used during development.

call.asp Using the CALL statement
datatypes.asp Testing VarType with literals
datetest.asp Testing variety of Date and Time functions.
extension.asp Testing a dotnet extension method
forloop.asp Testing a for loop in a asp page
if.asp Testing the if conditional
